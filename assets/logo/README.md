## ¿Por qué SVG?

- Se ve perfecto en cualquier tamaño (no se pixelea)
- Pesa muy poco (1-5 KB)
- Se puede colorear desde CSS si es necesario

## ¿Qué hago si solo tengo PNG?

Funciona también. Cambia las extensiones en los HTML:
```html
<!-- De esto: -->
<img src="../assets/logo/fractal-isotipo.svg">

<!-- A esto: -->
<img src="../assets/logo/fractal-isotipo.png">
```

Pero intenta conseguir los SVG — son mucho mejores para web.
